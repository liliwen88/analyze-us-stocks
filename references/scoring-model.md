# 量化评分模型

本文档定义了一个 **0-100 综合评分系统**，作为 `probability-framework.md` 中定性概率框架的量化补充。两个框架应配合使用，交叉校验。

---

## 一、总体评分结构

| 大类 | 权重 | 满分 | 子项数 |
|------|------|------|--------|
| 1. 基本面 | 25% | 25 | 4 |
| 2. 估值 | 15% | 15 | 3 |
| 3. 催化剂 | 20% | 20 | 4 |
| 4. 技术面 | 15% | 15 | 3 |
| 5. 情绪/仓位 | 15% | 15 | 4 |
| 6. 宏观/制度 | 10% | 10 | 3 |
| **总计** | **100%** | **100** | **21** |

---

## 二、详细评分规则

### 1. 基本面 (0-25 分)

#### 1.1 营收增长 (0-8 分)
| 分数 | 条件 |
|------|------|
| 8 | 营收增速 >20% YoY 且加速中（连续 3 季度递增） |
| 6 | 增速 10-20% 且稳定或加速 |
| 4 | 增速 0-10% 或减速但正向 |
| 2 | 增速负但接近拐点（指引转正、新产品周期） |
| 0 | 营收持续负增长，指引无改善 |

#### 1.2 盈利能力 (0-7 分)
| 分数 | 条件 |
|------|------|
| 7 | 毛利率/营业利润率扩张 + 利润率在行业中领先 |
| 5 | 利润率稳定或小幅扩张 |
| 3 | 利润率承压但有可见改善路径 |
| 1 | 利润率持续恶化 |
| 0 | 持续亏损，无盈利时间表 |

#### 1.3 财务健康度 (0-5 分)
| 分数 | 条件 |
|------|------|
| 5 | 净债务/EBITDA < 1x、FCF 充沛、流动性充足 |
| 3 | 负债率适中，FCF 覆盖利息 |
| 1 | 高杠杆、FCF 为负或勉强覆盖利息 |
| 0 | 流动性危机、即将违约或需要紧急融资 |

#### 1.4 成长驱动与护城河 (0-5 分)
| 分数 | 条件 |
|------|------|
| 5 | TAM 大且增长 + 宽护城河 + 结构性顺风 |
| 3 | 护城河中等，竞争格局稳定 |
| 1 | 护城河窄，竞争加剧 |
| 0 | 无护城河，商品化业务，市场份额流失 |

---

### 2. 估值 (0-15 分)

#### 2.1 绝对估值 (0-6 分)
| 分数 | 条件 |
|------|------|
| 6 | PE/EV 显著低于历史和行业均值（>30% 折价） |
| 4 | 估值低于均值 10-30% 或合理偏低 |
| 2 | 估值大致在均值+/-10% |
| 0 | 估值显著高于历史和行业（>30% 溢价） |

#### 2.2 成长调整估值 (PEG/FCF Yield) (0-5 分)
| 分数 | 条件 |
|------|------|
| 5 | PEG < 0.8 或 FCF Yield > 8% |
| 3 | PEG 0.8-1.5 或 FCF Yield 4-8% |
| 2 | PEG 1.5-2.5 或 FCF Yield 2-4% |
| 0 | PEG > 2.5 或无利润、FCF 为负 |

#### 2.3 利率敏感度 (0-4 分)
| 分数 | 条件 |
|------|------|
| 4 | 利率下行环境 + 低久期/低杠杆（双重受益） |
| 2 | 中性 |
| 0 | 利率上行环境 + 高久期/高杠杆（双重受压） |

---

### 3. 催化剂 (0-20 分)

#### 3.1 近期催化事件 (0-7 分)
| 分数 | 条件 |
|------|------|
| 7 | 多个强催化剂在 1-4 周内（财报、产品发布、FDA、监管决定） |
| 5 | 1 个明确重大催化剂 + 若干小催化剂 |
| 3 | 催化剂存在但不确定（时间未定、成功概率待验证） |
| 1 | 缺乏近期催化剂 |
| 0 | 近期负面事件（诉讼、调查、召回） |

#### 3.2 催化剂质量 (0-5 分)
| 分数 | 条件 |
|------|------|
| 5 | 可量化的高确定性催化剂（已宣布的涨价、已知的合同、专利到期） |
| 3 | 预期内的催化剂（正常财报节奏） |
| 1 | 模糊的希望（"未来可能有 AI 业务"） |

#### 3.3 中长期趋势顺风/逆风 (0-4 分)
| 分数 | 条件 |
|------|------|
| 4 | 强顺风：AI、降息、政策支持、行业上行周期 |
| 2 | 中性 |
| 0 | 强逆风：行业萎缩、监管打压、技术替代 |

#### 3.4 意外催化潜力 (0-4 分)
| 分数 | 条件 |
|------|------|
| 4 | 可能被低估的催化（拆分、被收购、隐藏资产） |
| 2 | 有一定的期权价值（新产品线试水、战略转型早期） |
| 0 | 无意外催化潜力 |

---

### 4. 技术面 (0-15 分)

#### 4.1 趋势 (0-6 分)
| 分数 | 条件 |
|------|------|
| 6 | 多个时间框架上行趋势（日/周/月线共振） + 均线多头排列 |
| 4 | 中期趋势向上，短期超买或盘整 |
| 2 | 趋势不明朗，横盘震荡 |
| 0 | 下行趋势明确，均线空头排列 |

#### 4.2 支撑/阻力位置 (0-5 分)
| 分数 | 条件 |
|------|------|
| 5 | 接近强支撑/突破阻力，平台突破 |
| 3 | 在区间内中性位置 |
| 1 | 接近强阻力/跌破支撑 |
| 0 | 创新低，下方无明显支撑 |

#### 4.3 量价关系 (0-4 分)
| 分数 | 条件 |
|------|------|
| 4 | 上涨放量/下跌缩量，量价健康 |
| 2 | 量价中性 |
| 0 | 上涨缩量/下跌放量，量价背离 |

---

### 5. 情绪/仓位 (0-15 分)

#### 5.1 分析师情绪 (0-4 分)
| 分数 | 条件 |
|------|------|
| 4 | 多家上调评级/目标价，且上调趋势持续 |
| 2 | 评级稳定，无明显变化 |
| 0 | 多家下调，下调趋势持续 |

#### 5.2 仓位拥挤度 (0-4 分)
| 分数 | 条件 |
|------|------|
| 4 | 低拥挤（机构持仓低、空头低），有加仓空间 |
| 2 | 中等拥挤 |
| 0 | 高拥挤（基金重仓、热搜第一），资金面撤离风险 |

#### 5.3 内部人/回购信号 (0-4 分)
| 分数 | 条件 |
|------|------|
| 4 | 内部人集群买入 + 公司积极回购减股 |
| 2 | 适度回购或中性内部人交易 |
| 0 | 内部人密集卖出 + 回购停止/SBC 稀释 |

#### 5.4 散户/社交情绪 (0-3 分)
| 分数 | 条件 |
|------|------|
| 3 | 散户关注度上升 + 正面情绪（潜在动量）但未过热 |
| 1 | 中性 |
| 0 | 过度炒作（泡沫风险）或普遍悲观但无催化剂（价值陷阱） |

---

### 6. 宏观/制度 (0-10 分)

#### 6.1 利率与货币政策 (0-4 分)
| 分数 | 条件 |
|------|------|
| 4 | 利率下行/宽松周期 + 股票友好政策 |
| 2 | 中性/混合信号 |
| 0 | 利率上行/紧缩 + 鹰派美联储 |

#### 6.2 经济周期位置 (0-3 分)
| 分数 | 条件 |
|------|------|
| 3 | 扩张早期/中期 |
| 2 | 周期成熟/晚期 |
| 1 | 衰退风险/早期衰退 |
| 0 | 明显衰退/危机 |

#### 6.3 行业/风格适配 (0-3 分)
| 分数 | 条件 |
|------|------|
| 3 | 当前宏观环境直接利好该行业/风格 |
| 2 | 中性 |
| 0 | 当前宏观环境严重不利（如加息对高杠杆REITs） |

## 三、综合评分区间与操作映射

| 评分区间 | 标签 | 含义 | 对应操作 |
|---------|------|------|---------|
| **85-100** | 强烈看涨 | 多维度强信号共振 | 积极买入/加仓 |
| **70-84** | 看涨 | 多数信号偏多，少数中性 | 买入/加仓，设置止损 |
| **55-69** | 偏涨 | 看涨信号占优但有关注点 | 持有/观望，回调买入 |
| **45-54** | 中性 | 多空信号大致平衡 | 持有/观望，等待催化剂 |
| **30-44** | 偏跌 | 看空信号占优但有支撑 | 减仓/观望，不抄底 |
| **15-29** | 看跌 | 多数信号偏空 | 减仓/回避 |
| **0-14** | 强烈看跌 | 多维度危险信号共振 | 果断回避/做空（高级） |

## 四、一致性校验规则

### 4.1 评分-概率冲突检测

如果量化评分与概率框架的结论出现以下冲突，必须触发复核：

| 评分 | 概率判断 | 冲突? | 处理 |
|------|---------|-------|------|
| >70 (看涨) | 看涨概率 < 50% | **冲突** | 复核评分和概率，找出分歧原因 |
| <30 (看跌) | 看跌概率 < 45% | **冲突** | 同上 |
| 45-54 (中性) | 看涨或看跌 > 60% | **冲突** | 同上 |

### 4.2 复核流程

1. **识别分歧来源**：哪个大类评分与概率判断最不一致？
2. **检查权重**：该大类在当前情境下是否被过度/低估？
3. **检查数据质量**：分歧是否因为数据缺失或过时？
4. **重新评估**：调整评分或概率，直到两者方向一致
5. **记录调整**：在报告中简要说明一致性校验结果

### 4.3 信心区间调整

根据数据完整性自动调整分数置信度：

| 数据完整性 | 调整方式 |
|-----------|---------|
| 完整 | 分数不变，置信度高 |
| 部分缺失 | 分数不变，置信度降为中；标注哪些子项因数据不足而估计 |
| 严重缺失 | 分数下调 10%，置信度降为低；明确声明不适用 |

## 五、使用说明

### 与概率框架的关系

- **概率框架** (`probability-framework.md`)：自上而下，基于场景的定性判断
- **评分模型** (本文档)：自下而上，基于 21 个子项的定量评估
- **两者必须方向一致**：如评分 > 70 但概率看涨 < 50%，触发复核

### Agent 使用指南

1. 在完成数据采集和交叉验证后，进入评分
2. 逐项评分，使用高（≥2 个来源确认）、中（单源但有信心）、低（估计值）标注每个子项的信心
3. 计算加权总分
4. 与概率框架的牛/中/熊概率对比，执行一致性校验
5. 在报告中呈现总分和各大类得分（简化版），完整评分表放在附录或视需求展示

### 评分快速参考（简版）

当时间有限时，Agent 可以仅对 6 大类评分（跳过 21 子项），使用下表：

| 大类 | 权重 | 快速评分 (0-10) |
|------|------|----------------|
| 基本面 | 25% | |
| 估值 | 15% | |
| 催化剂 | 20% | |
| 技术面 | 15% | |
| 情绪/仓位 | 15% | |
| 宏观/制度 | 10% | |

总分 = 加权平均 × 10，结果区间 0-100。
